# Automation Framework - OrangeHRM

## Tech Stack
- Selenium WebDriver
- Java
- TestNG
- Cucumber (BDD)
- Maven
- Extent Reports

## Framework Features
- Page Object Model (POM)
- Explicit Wait Handling
- Screenshot on Failure
- Extent Reporting
- Data Driven Testing (Excel)
- CI/CD Integration (Jenkins)

## Project Structure
- base
- pages
- utils
- stepdefinitions
- runners
- features

## How to Run
1. Clone the repo
2. Run TestRunner.java
