# 🚀 Automation Framework - OrangeHRM

## 📌 Overview
This project is a Hybrid Automation Framework built using Selenium, Java, TestNG, and Cucumber (BDD) to automate the OrangeHRM application.

---

## 🛠 Tech Stack
- Selenium WebDriver
- Java
- TestNG
- Cucumber (BDD)
- Maven
- Extent Reports
- Git & Jenkins

---

## 🔥 Features
- Page Object Model (POM)
- Explicit Wait Handling
- Screenshot on Failure
- Extent Reporting
- Data Driven Testing (Excel)
- BDD Framework (Cucumber)
- CI/CD Integration (Jenkins)

---

## 📁 Project Structure
- base → Driver setup & teardown
- pages → Page classes (POM)
- utils → Reusable utilities
- stepdefinitions → Cucumber steps
- runners → Test execution
- features → Gherkin scenarios

---

## ▶️ How to Run
1. Clone the repository
2. Open project in IDE
3. Run TestRunner.java

---

## 📊 Reporting
- Extent Reports generated after execution
- Screenshots captured on failure

---

## ⚙️ CI/CD
- Integrated with Jenkins for automated test execution
